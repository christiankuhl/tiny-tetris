# Tiny Tetris

## About

A Tetris implementation that "fits on a business card" (30 lines x 80 characters)

## Installation

```bash
git clone https://github.com/christiankuhl/tiny-tetris
cd tiny-tetris && python3 tiny-tetris.py
```
## Controls

up - rotate

left - move left

right - move right

down - move down

q - quit
